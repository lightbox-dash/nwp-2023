作品規範

  • 投件作品，須為 2019 年後之創作。
  • 投件作品，須未曾獲得或入選國內外之攝影或藝術創作獎項。
  • 個人或團體，至多可各投一組作品參加。
  • 投件作品，可為進行中，或已完成之創作。

投件須知

  1. 評審會議結束後，所有投件作品將以去識別的方式，在線上展覽頁面公開展示（僅呈現作品之名稱、簡介和圖檔，不揭露姓名等個資），提供創作獎之所有參加者和贊助者觀摩與交流，展期約 1 個月。
  2. 參加者應對其所提供之資料及投件作品之真實性負責，且不得侵害他人之著作權等相關權利。否則除應依法自負法律責任外，主辦單位得取消其獲獎資格，並追回已頒發之獎金和獲獎證書，且下一屆不得參加。
  3. 若投件作品之相關資料含有任何他人創作之內容，請自行於送件前取得相關授權同意。若投件作品及所提供之相關資料，事後涉及相關法律爭議，由參加者自行負責。
  4. 主辦單位為教育、研究及推廣之目的，得就投件作品舉辦實體及線上之公開展覽、頒獎典禮及作品發表會，開放大眾及媒體攝（錄）影，主辦單位並得就投件作品出版發行實體及數位之展覽圖錄及文宣，進行實體及線上之發表。參加者同意主辦單位依上開行為所需，將其著作財產權非專屬授權主辦單位為不限時地之利用。
  5. 參加者同意，如作品獲得獎項，主辦單位將揭露其姓名或別名，參加者並同意主辦單位就獲獎作品出版發行實體及數位專輯，舉行實體及線上之公開發表及展覽，期間將開放大眾及媒體攝（錄）影。參加者並同意主辦單位依上開行為所需，將其著作財產權非專屬授權主辦單位為不限時地之利用。
